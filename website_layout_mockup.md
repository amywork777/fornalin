## Mobile Layout Adaptation

On mobile devices, the layout will stack vertically with:

1. Hero section (simplified animation)
2. Date overview cards (stacked vertically)
3. Vertical timeline instead of map
4. Simplified about section
5. Gallery as a swipeable carousel
6. RSVP buttons stacked but larger for easy tapping

All interactive elements will be optimized for touch interaction rather than hover states.
