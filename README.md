# Nalin's Capybara Adventure Invitation

A whimsical and interactive web invitation for a day of exploration in San Francisco, featuring capybaras, purple discoveries, and hummus delights.

## Features

- Interactive map of San Francisco with highlighted locations
- Animated capybara characters throughout the experience
- Custom cursor with paw print trail
- Responsive design for all devices
- Light and dark theme toggle
- RSVP functionality

## Technologies

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- Confetti.js for celebratory effects

## Setup

This is a static website that can be deployed on any hosting platform. No build process or server-side code is required.

1. Clone the repository
2. Open index.html in your browser to view locally

## Live Demo
