# Capybara Illustrations Specification

This document outlines the illustrations needed for the capybara-themed date invitation website. Each illustration has been designed to support the purple color scheme, hummus elements, and create a charming yet sophisticated aesthetic.

## Character Illustrations

### 1. Hero Capybara
- **Style:** Friendly, detailed illustration with slightly cartoon-like proportions
- **Pose:** Standing upright, smiling and waving with one paw
- **Attire:** Purple bow tie
- **Details:** 
  - Warm brown fur with subtle texture
  - Expressive, friendly eyes
  - Slightly rounded proportions for added cuteness
- **Animation Requirements:** 
  - Separate layers for head, body, arms, and eyes for animation
  - Additional arm position for waving animation
  - Blinking eye states
- **Dimensions:** 400px × 500px
- **Format:** SVG for animation and scaling

### 2. Guide Capybara
- **Style:** Smaller, simpler version of main character
- **Pose:** Standing upright with helpful posture
- **Attire:** Purple tour guide hat and small backpack
- **Details:**
  - Simplified features but recognizably the same character
  - Friendly expression
  - Small notepad or map in paw
- **Animation Requirements:**
  - Separate head layer for looking around
  - Arm layer for pointing gestures
  - Different expression states (happy, thinking, explaining)
- **Dimensions:** 150px × 200px
- **Format:** SVG or sprite sheet

### 3. Map Route Capybara
- **Style:** Tiny, simplified chibi version
- **Pose:** Walking pose with visible legs
- **Attire:** Tiny purple backpack
- **Details:**
  - Extremely simplified but instantly recognizable as capybara
  - Exaggerated walking motion
- **Animation Requirements:**
  - Walking cycle (4-6 frames)
  - Celebration pose when reaching destinations
- **Dimensions:** 60px × 60px
- **Format:** Sprite sheet or small SVG set

### 4. RSVP Capybara
- **Style:** Medium detail, highly expressive
- **Pose:** Holding a "Will you join me, Nalin?" sign
- **Attire:** Optional purple party hat
- **Details:**
  - Hopeful expression
  - Sign designed like a purple invitation card
  - Slightly eager forward-leaning posture
- **Animation Requirements:**
  - Happy dance pose for "Yes" response
  - Nodding understanding pose for "Need to check" response
- **Dimensions:** 300px × 400px
- **Format:** SVG with separate layers for animation

## Technical Requirements

### 1. Style Consistency
- All illustrations should maintain consistent stylistic elements:
  - Same eye style and proportions across characters
  - Consistent fur texture and coloration
  - Uniform line weight and corner rounding

### 2. Color Specifications
- Primary capybara color: #A47551 (warm brown)
- Highlight fur color: #C4946A (lighter brown)
- Shadow fur color: #8A5D3B (darker brown)
- Purple palette for accessories: #663399, #9370DB, #D8BFD8
- Hummus colors: #F5F5DC (base), #708238 (herbs/olive oil)

### 3. File Organization
- Each illustration delivered with:
  - Source file with layers (AI or PSD)
  - Optimized SVG (where applicable)
  - PNG with transparency at 1x and 2x resolutions
  - Documentation for animation-ready illustrations

### 4. Accessibility Considerations
- All illustrations should work without color (for colorblind users)
- Important elements should have sufficient contrast
- Animation-ready illustrations need logical layer naming for developer implementation

This specification provides a comprehensive guide for creating the capybara illustrations needed for the website, ensuring they support the purple color scheme, incorporate hummus elements where appropriate, and maintain a cohesive, charming aesthetic throughout the site.
